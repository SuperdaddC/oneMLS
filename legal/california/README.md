## Recent Legislative Changes (2024-2025)

### AB 2992 -- Buyer-Broker Representation Agreements (Effective Jan 1, 2025)
- Mandates written buyer-broker agreements for all property types
- 90-day maximum duration (3 months)
- Prohibits automatic renewal
- Must specify compensation, services, and expiration date
- See: [Buyer Representation](buyer-representation.md)

### AB 1345 -- Residential Exclusive Listing Agreements Act (Effective Jan 1, 2024)
- 24-month maximum for exclusive listing agreements on single-family residential
- 12-month maximum for renewals
- Prohibits recording with county recorder
- Prohibits automatic renewal
- See: [Listing Agreement](listing-agreement.md)

### NAR Settlement Changes (Effective Aug 17, 2024)
- MLS may no longer display buyer agent compensation offers
- Written buyer-broker agreement required before touring
- Compensation negotiated off-MLS or via purchase agreement
- See: [Commission Law](commission-law.md)

### AB 968 -- Property Flipping Disclosure (Effective Jul 1, 2024)
- Sellers who flip 1-4 unit properties within 18 months must disclose repairs/renovations over $500
- Must provide contractor names and contact info
- See: [Seller Disclosures](seller-disclosures.md)

## Penalties for Non-Compliance

| Violation | Penalty |
|---|---|
| Failure to deliver TDS | Actual damages (Civil Code 1102.13) |
| Willful/negligent TDS violation | Actual damages + potential punitive damages |
| Agency disclosure failure | DRE disciplinary action, license suspension/revocation |
| Listing agreement violations (AB 1345) | Agreement void and unenforceable; DRE discipline |
| Buyer-broker violations (AB 2992) | Agreement void and unenforceable; DRE discipline |
| Fair housing violations (FEHA) | Civil penalties up to $42,500 (first), $127,500 (subsequent); actual damages |
| BPC 10176/10177 violations | License revocation/suspension; fines |
| Escrow law violations | Criminal penalties; license revocation |

## Architecture Notes for OneMLS Contract Builder

### Compliance Engine Requirements
1. **Jurisdiction detection** -- Determine applicable local ordinances by property address/county
2. **Property type routing** -- Different disclosures for 1-4 unit residential vs. commercial vs. land
3. **Exemption checking** -- Probate, foreclosure, court-ordered transfers exempt from TDS
4. **Timeline enforcement** -- Track disclosure delivery dates and rescission windows
5. **Form generation** -- Generate or reference appropriate C.A.R. forms per transaction type
6. **Signature tracking** -- Track delivery and acknowledgment signatures for all disclosures
7. **Amendment handling** -- Support material amendments with new rescission windows
